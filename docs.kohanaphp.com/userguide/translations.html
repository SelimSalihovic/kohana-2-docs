Translations

Translations of the user guide aren't currently set up, but they will work as follows:

The root namespace is in english. Other languages will copy the namespace structure of the root but will reside in their own namespace using their language code (and localization if needed).

For example, the French translation of general/configuration would be in fr/general/configuration.

On each page there will be a list of all available languages, as well as an indication of which languages the current page has been translated to.

Translations of Kohana

Translations of Kohana start off by translating the English i18n files in the system/i18n directory to the language of choice. For more information see the forum.
